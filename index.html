About Me

I am a fullstack web and mobile developer with an eye for creating responsive websites and apps
For the past one year, I've been developing applications for the web and I am currently learning cross platform mobile apps development using React Native. In my spare time, I contribute to Open Source projects and write blogs.
And if I want to wind down, I like to listen to music, play games, table tennis, badminton, watch football/soccer, and go to the gym.

Skills
